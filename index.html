Algunos de mis proyectos recientes

Build Your Body

Aplicación web para la creación de rutinas fitness, con base de datos en MongoDB, desarrollada en Node.js y Angular y con un diseño totalmente responsive para mantener una experiencia óptima en cualquier dispositivo. Permite a los usuarios registrados crear rutinas personalizadas, buscar ejercicios por grupo muscular y acceder a videos explicativos de cada uno.

Personajes Futurama

Aplicación web desarrollada en React que muestra personajes obtenidos de una base de datos. Permite a los usuarios buscar personajes fácilmente a través de una barra de búsqueda interactiva, mejorando la experiencia de exploración.

Echale un vistazo!

Caldera Restaurante

Diseñé una página web para un restaurante con HTML y CSS, incorporando una interfaz intuitiva que incluye una sección de reservas en línea y la integración con Google Maps para facilitar la ubicación del local. Implementé animaciones y efectos visuales sutiles.
